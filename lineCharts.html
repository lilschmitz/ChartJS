Line Charts

This simple Line Chart plots one dataset. The dataset consists of an array of datapoints (data) A colors for the columns(backgroundColor) And a label for the dataset (label)

Outside of the dataset the columns labels are specified.

Click 'Add Data' to plot a new datapoint

{
    datasets: [{
        data: [
            11,
            16,
            7,
            3,
            14
        ],
        backgroundColor: "rgba(220,220,220,0.5)",
        label: 'My dataset' // for legend
    }],
    labels: [
        "Red",
        "Green",
        "Yellow",
        "Grey",
        "Blue"
    ]
}
